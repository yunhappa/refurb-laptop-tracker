\# 리퍼 트래커 | Refurb Laptop Tracker



리퍼 트래커는 리퍼·중고 노트북의 가격 데이터를 수집하고, 모델·사양·가격 정보를 비교하여 사용자가 현재 가격이 구매할 만한 수준인지 판단할 수 있도록 돕는 웹서비스입니다.



공개 서비스 주소:

https://refurb-laptop-tracker.onrender.com



\## 프로젝트 개요



리퍼·중고 노트북은 같은 모델이라도 판매처, 사양, 상태, 가격 차이가 커서 사용자가 적정 가격을 판단하기 어렵습니다.

이 프로젝트는 네이버 쇼핑 API를 통해 리퍼·중고 노트북 데이터를 수집하고, RAM·SSD·CPU·가격·판매처 수·가격 이력 등을 종합해 구매 판단 정보를 제공합니다.



\## 주요 기능



\* 리퍼·중고 노트북 데이터 수집

\* RAM, SSD, CPU 사양 자동 추출

\* 실사용 후보 상품 필터링

\* 모델·사양별 중복 그룹화

\* 가성비 점수 계산

\* 구매 추천 / 구매 고려 / 보류 판단

\* 평균가, 최저가, 관측 수 기반 구매 적기 점수 제공

\* 상품 페이지 외부 연결

\* 모바일 화면 대응



\## 기술 스택



\* Python

\* Flask

\* HTML / CSS

\* CSV 기반 데이터 저장

\* Naver Shopping Search API

\* GitHub

\* Render



\## 데이터 처리 흐름



1\. 네이버 쇼핑 API를 통해 리퍼·중고 노트북 상품 데이터를 수집합니다.

2\. 상품명에서 RAM, SSD, CPU 정보를 추출합니다.

3\. 가격, 사양, 판매처 정보를 기준으로 실사용 후보를 필터링합니다.

4\. 동일 모델·사양 상품을 그룹화합니다.

5\. 사양 점수와 가격을 비교해 가성비 점수를 계산합니다.

6\. 평균가, 최저가, 관측 수, 판매처 수를 종합해 구매 적기 점수를 산출합니다.

7\. Flask 웹앱에서 검색 결과와 추천 정보를 제공합니다.



\## 현재 버전



현재 버전은 CSV 기반 프로토타입입니다.

데이터가 장기간 누적되면 가격 이력, 평균가, 최저가, 가격 하락률을 더 정교하게 반영할 수 있습니다.



\## 향후 개선 방향



\* 데이터 자동 수집 스케줄러 구축

\* CSV 기반 저장 방식에서 데이터베이스로 전환

\* ClickHouse 기반 대용량 가격 이력 분석

\* 가격 변동 그래프 제공

\* 관심 모델 저장 기능

\* 가격 하락 알림 기능

\* 판매처 신뢰도 반영

\* 광고 및 제휴 모델 검토



\## 주의 사항



본 서비스는 수집된 공개 쇼핑 데이터와 자체 계산 기준을 바탕으로 구매 판단 정보를 제공합니다.

실제 구매 여부는 사용자가 상품 상태, 판매처, 배송 조건, 보증 여부 등을 직접 확인한 뒤 결정해야 합니다.



\## 개발 목적



이 프로젝트는 Edge Cloud / 데이터 분석 수업 프로젝트에서 시작되었으며, 이후 실제 웹서비스 형태로 발전시키기 위해 Flask와 Render를 활용해 공개 배포했습니다.
